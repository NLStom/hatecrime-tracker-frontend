Install library

```bash
npm install
# or
yarn install
```

run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


Techstack: Front-end: React, Next.js, react-map-gl, react-chartjs-2, tailwindcss
